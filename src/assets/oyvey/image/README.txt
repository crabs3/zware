Put Chams image-mode textures in this folder.

Examples:
- img.png
- camo.jpg

In the Chams Image setting, use just the filename, for example:
- img.png

Plain filenames now resolve to:
- assets/oyvey/image/<filename>

Full resource ids still work too, for example:
- oyvey:image/img.png
- minecraft:textures/misc/enchanted_glint_item.png
